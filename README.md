# 🧭 Y-Compass (와이컴퍼스)
연세대 AX 캠프 Track 1 — 소그룹 챌린지  
**근거 기반 AI 진학 카운셀러(Streamlit MVP+)**: 3분 입력만으로 **A/B/C(안정/적정/도전)** 진학 경로와 **8주 준비 로드맵**을 제시합니다.  
> ⚠️ 본 앱은 **합격/불합격 “확률 단정”을 하지 않으며**, 항상 **구간(안정/적정/도전)** + **근거/한계**를 함께 제공합니다.

## ✅ 핵심 요약 (제출용)
- **문제**: 입시 정보 과잉·분산, 컨설팅 비용/접근성 장벽, “카더라” 조언으로 인한 신뢰성 부족  
- **해결**: **짧은 입력 → 근거 기반 추천(출처/연도) → 즉시 실행 로드맵(8주)**  
- **차별점**: CSV 업로드로 **데이터 커버리지**를 확장하고, 데이터가 있을 때만 **연도별 기준선 시각화** 및 **설명가능 점수화(A/B/C)**를 제공합니다.

## ✨ 주요 기능
### 1) 8문항 진학 상황 스캔(Intake & Profiling)
- 성적(내신/모의) 구간, 희망 전형(수시/정시 및 수시 세부), 전공군/성향, 비교과, 제약, 목표 우선순위 입력
- 출력: **상황 요약(5줄)** + 강점/제약 중심으로 구조화된 입력(JSON)

### 2) CSV 업로드 기반 “데이터 커버리지” + 근거 생성
- 사용자가 업로드한 **대학/학과/전형/연도별 기준선**으로 근거 컨텍스트(evidence)를 자동 생성
- 결과 카드에 **커버리지 배지** 표시
  - **데이터 기반 ✅**: 업로드된 데이터 매칭 성공 → 근거/그래프 제공
  - **가이드 기반 🟡**: 매칭 실패(데이터 없음) → 수치 단정 없이 전략 가이드 제공

### 3) A/B/C(안정/적정/도전) “설명가능” 점수화
- 점수는 0~100, 요소별 가중치를 공개(사이드바에서 조정 가능)
  - **학업(성적)** / **비교과** / **전형-성향 적합도** / **제약(감점)**
- 총점에 따라 구간 자동 분류:
  - 75+ 안정 / 55~74.9 적정 / 0~54.9 도전  
- “왜 A/B/C인지”를 **기여도 breakdown**으로 설명

### 4) 데이터 기반일 때 연도별 미니 표/그래프(시각화)
- (예) 사용자 내신 vs 업로드된 **연도별 기준선(threshold)**을 한 화면에서 비교
- **line chart** + 표로 근거를 직관적으로 제시(심사자 설득력 강화)

### 5) OpenAI API(선택) / 키 없으면 룰베이스 데모 가능
- API Key가 있으면: OpenAI Responses API로 **A/B/C 추천 + 8주 로드맵**을 더 자연어 컨설팅 형태로 생성
- Key가 없으면: **rule-based** 플랜으로도 동작(데모/발표 가능)

## 🧩 사용 방법(How to Use)
1. **[데이터 업로드 탭]**에서 CSV 업로드(선택)  
2. **[진단 입력 탭]**에서 전형/성적/성향/제약 입력 후 `결과 생성`  
3. **[결과 탭]**에서
   - 커버리지 배지(데이터 기반/가이드 기반)
   - 가능성 카드(안정/적정/도전)
   - A/B/C 카드(추천 이유/액션/리스크)
   - 8주 로드맵(주차별 목표/할일/산출물)
   - (데이터 기반이면) 연도별 기준선 그래프 확인  
4. 필요 시 `결과 리포트(.json)` 다운로드
